# Del papel al digital

Proyecto de análisis de datos sobre la transformación de la escritura en la era digital.

## Objetivo

Analizar la disminución del uso del papel gráfico y compararlo con el uso digital entre 2016 y 2024.

## Herramientas utilizadas

- Python
- Pandas
- Matplotlib

## Datasets

- FAOSTAT
- Eurostat

## Proceso

1. Limpieza de datos
2. Transformación y unión de datasets
3. Visualización de tendencias

## Reflexión

Este proyecto nace como una reflexión sobre cómo ha cambiado nuestra forma de escribir y comunicarnos.
